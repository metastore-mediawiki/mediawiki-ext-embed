# Информация

Добавление встроенного содержимого в статью.

## Синтаксис

```
{{#embed: [URL]}}
```

## Установка

1. Загрузите папки и файлы в директорию `extensions/MW_EXT_Embed`.
2. В самый низ файла `LocalSettings.php` добавьте строку:

```
wfLoadExtension( 'MW_EXT_Embed' );
```
